# chat

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### For express server

```
npm run dev
```

### Pocketbase server

```
./pocketbase serve
```

### selenium-webdriver

```
For selenium to open browser need to change PATH in test.js:
const service = new chrome.ServiceBuilder(
  'C:/PATH/TO/DIVER/chromedriver.exe'
);
```
